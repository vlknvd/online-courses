## Особенности проекта

- Полностью адаптивный дизайн (320px → 4K)
- Фильтры по категориям с счётчиком
- Умный поиск по названию и автору (с дебаунсом)
- Чистый, читаемый и масштабируемый код

## Технологии

- HTML5 · SCSS · Vanilla JavaScript (ES6+)
- Адаптив без медиазапросов там, где это возможно
- Модульный JS (импорты, экспорты)
- Pixel-perfect вёрстка
